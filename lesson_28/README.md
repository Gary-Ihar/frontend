# Rspack project

## Setup

Install the dependencies:

```bash
npm install
```

## Get started

Start the dev server, and the app will be available at <http://localhost:8080>.

```bash
npm run dev
```

## Learn more

Прилага заметок

1. Создавать Заметки +
2. Редактировать Заметки + (редактировать имя)
3. Удалять Заметки +
4. Двигать Заметки + (последняя оставалась бы наверху)
5. Хранить Локал стораге + (хранить контент в локал сторадже и изменять его)
6. нормально отрефакторить index.js
